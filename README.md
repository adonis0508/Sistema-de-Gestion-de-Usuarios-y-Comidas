# Sistema de Gestión Asistido por IA - Casino de Oficiales GECB

Una plataforma web integral diseñada para optimizar y automatizar la administración diaria del Casino de Oficiales de la Guarnición Ejército City Bell (GECB). 

Esta aplicación fue impulsada originalmente mediante **Google AI Studio** y se encuentra desplegada en Google Cloud Run. Su propósito es modernizar la logística del casino integrando bases de datos de usuarios y un control estricto de roles, agilizando tareas que van desde la toma de asistencia hasta la planificación de menús.

## 🚀 Características Principales

* **Gestión Inteligente de Menús:** Organización y configuración dinámica de los menús ofrecidos en el casino.
* **Control de Asistencia:** Sistema integrado para registrar y monitorear la asistencia del personal de manera eficiente.
* **Base de Datos de Usuarios:** Registro estructurado de todo el personal que interactúa con el sistema.
* **Control de Acceso Basado en Roles (RBAC):** La plataforma adapta su interfaz y permisos según el tipo de usuario autenticado:
    * **Comensal:** Visualización de menús y opciones básicas de usuario.
    * **Mozo:** Herramientas orientadas al servicio, atención y registro de mesas.
    * **Administrador de Mesa:** Privilegios de gestión, supervisión de asistencia y configuración del sistema.

## 🛠️ Tecnologías Utilizadas

* **Google AI Studio / Gemini API**
* **Google Cloud Run:** Despliegue de la aplicación (Serverless) garantizando alta disponibilidad y escalabilidad.
* **Base de Datos:** Firestore, Firebase Authentication. 
* **Frontend/Backend:** JavaScript, Firebase, TypeScript, JS, NodeJS.
